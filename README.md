# Meal-Plan-Generator
C++ program that generates and validates meal plans based on calorie goals and TDEE calculations

## Features
- Calculate TDEE using the Harris-Benedict equation
- Generate randomized meal plans
- Create custom meal plans by selecting from a preloaded meal lists for breakfast, lunch, and dinner using structured data
- Compare total calories against a user-defined calorie goal or TDEE
- Save meal plans to a file
- Read saved meal plans from a file

## Tools & Concepts
- C++
- STL vector
- File (ofstream/ifstream)

## Project Structure
- 'main.cpp': menu-driven and user interaction
- 'functions.cpp': conatins function implementations
- 'functions.hpp': contains function declarations and 'Meal' struct
